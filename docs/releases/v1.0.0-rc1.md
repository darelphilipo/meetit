# v1.0.0-rc1 — Pitch Lifecycle + Daily Cleanup + UI Polish

First release candidate. Combines 7 PRs that were developed and verified individually into a single shippable bundle. This is the candidate for the first public release of the app.

## What's in this release

### For community members

- **💡 Pitch a meetup** — anyone can submit an idea for a meetup. The idea is queued for a moderator to review. The submitter gets a private message confirming the submission and is told to check My Stuff for status.
- **📊 My Stuff status tracking** — when a moderator acts on your pitch, you see a clear status line in My Stuff: "Pending review" while waiting, "Approved" (with a link to submit it as an event) when the mod says yes, or "Dismissed" (with the moderator's reason) when the mod says no.
- **🧹 Aged data cleanup** — events and pitches older than the configured threshold (default 30 days) are automatically hard-deleted by a daily background job. The app stays tidy without manual effort.

### For organizers

- **📣 Automatic announcement post on approval** — the moment a moderator approves your event, a public post is created on the subreddit with the full event details. The community can start commenting, planning, and coordinating rides immediately, not 24 hours before the event.
- **📅 Google Calendar + Maps links in every post** — reminder posts and the announcement post include "Add to Google Calendar" and "Open Map" links. The map field accepts Google Maps, Zoom, Google Meet, or any other link.
- **🔗 One-tap deep link from posts** — the reminder and announcement posts link back to the app. Users land directly in the RSVP flow.
- **🛡️ Auto-RSVPed as organizer** — when your event is approved, you're automatically on the attendee list. You appear in My Stuff → RSVPs immediately.

### For moderators

- **🛡️ Single-screen mod dashboard** — the mod dashboard has three tabs: Pending (events to review), Published (your approved events), and Pitches (community-submitted ideas). One click from the home page.
- **✅ / 🗑️ Approve or dismiss a pitch in one tap** — no more navigating through forms. Click the green Approve button or the Dismiss button on a pitch card. The pitcher gets a private message automatically. The pitch disappears from your queue.
- **🧹 Aged cleanup with manual override** — a small broom icon in the mod dashboard header lets you run the cleanup on demand (e.g. before a reset or a fresh release). The threshold is configurable per install.
- **🐛 Debug log panel (opt-in)** — a 🐛 button appears for mods only when the `show_debug_panel` install setting is enabled. It shows server logs and client logs merged into a single scrollable panel. Default is OFF so the public app is clean.

### For subreddit admins

- **⚙️ App Installation Settings** — five settings to tune the app per subreddit:
  - `primary_color`, `secondary_color`, `use_brutalist_borders` — visual theme
  - `mod_usernames` — comma-separated list of mods (the mod gate)
  - `reminder_hours`, `reminder_hours_2` — when to post reminders (default 24h and 2h before an event)
  - `timezone` — the community's timezone for event times and reminders
  - `show_debug_panel` — whether the in-app debug panel is visible to mods
  - `cleanup_after_days` — how long to keep events and pitches before auto-deletion (default 30)
  - `pause_cleanup` — freeze the daily auto-cleanup (manual button still works)
- **📌 Launcher post deep linking** — when a mod creates a "Meetit - Community Meetups" launcher post via the subreddit menu, all reminder and announcement posts link back to that post. The community lands directly in the app.

## What was tested

- **110 unit tests** for pure helpers, all passing
- **48 OpenSpec specs** validated
- **Manual playtest in production** for the pitch feedback loop (the highest-risk feature): two devices, one submitting and one dismissing, confirmed the dismissal, the status line in My Stuff, and the cross-filter counts all work end-to-end
- **Build and type-check** clean

## What was NOT in this release

These were intentionally deferred to post-launch:
- Category-based filtering on the home page
- "→ Event" conversion flow (one-click promotion of an approved pitch to an event)
- Public Ideas surface (browse all community pitches)
- Voting on pitches
- Per-bucket cleanup thresholds (different thresholds for events vs. pitches)
- Archive view for soft-deleted items

## Upgrade notes

This is the first RC. There's no upgrade path from a previous version. Fresh install on a subreddit will:
- Add the 5 install settings with their defaults
- Add the daily CRON task at 03:00 UTC
- Add the mod dashboard accessible from the home page header (mods only)
- Show a 🎉 Meetit logo and "Real meetups. Real people." tagline on the home page

## Known limitations

- The `u/username` mentions in the going list trigger Reddit notifications (capped at 20 names per post). RSVPing is consent to appear in the public list.
- The `Open in Meetit to RSVP` deep link points to the most recent launcher post the mod created. If no launcher post exists, it falls back to the subreddit homepage.
- The aged cleanup CRON runs at 03:00 UTC. Manual button in the mod dashboard header can trigger it on demand.
