## 1. What the App does

Meetit is a community-driven event management tool for Reddit. It allows subreddit members to:

- Browse upcoming community events pinned to the subreddit home
- RSVP to events (optionally sharing contact info with the organizer)
- Submit pitches for new meetup ideas
- Receive reminder posts before events
- Add events to their Google Calendar
- Share "I'm going" threads

Subreddit moderators can additionally:

- Approve or decline submitted events
- Approve, dismiss, or delete community pitches
- Export attendee lists to CSV
- View server logs (if the `show_debug_panel` setting is enabled)
- Configure app behavior through subreddit-level settings

Subreddit administrators can additionally:

- Install or remove the App from their subreddit
- Configure all 9 App Install Settings (brand colors, mod list, timezone, reminder windows, debug panel, cleanup threshold, cleanup pause)
